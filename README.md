### cute-qq-redpoint
SVG实现新版手机QQ的手势消除红点消息的动画Demo
小Demo，目前还有一些小bug，基本效果是出来了。
可以打开手机qq，在底部菜单出现红点时体验一下。

### 来源及所用知识
前几天看到一个Android原生版本的，手痒就用js做了一个
勾股定理，简单几何，svg, 二次贝赛尔曲线
其实，只要将四个点的坐标拿到，接下来就好办了
